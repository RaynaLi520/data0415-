# FAAM 自动化系统配置指南

## 系统概述

FAAM自动化系统会在**每天北京时间18:30**自动执行以下任务:
1. 爬取Target网站A New Day和Wild Fable品牌的女装商品
2. 将爬取的数据导入SQLite数据库
3. 自动识别并标记新品
4. 生成统计报告

## 文件说明

### 核心脚本
- `auto_crawler.py` - 自动化爬虫,爬取Target商品数据
- `auto_import.py` - 数据导入器,将数据导入数据库并标记新品
- `auto_scheduler.py` - 调度器,协调爬虫和导入器的执行

### 批处理脚本
- `setup_task.bat` - 安装Windows任务计划(需要管理员权限)
- `run_now.bat` - 立即手动执行一次完整任务

### 目录结构
```
FAAM/
├── auto_crawler.py        # 爬虫脚本
├── auto_import.py         # 导入脚本
├── auto_scheduler.py      # 调度脚本
├── setup_task.bat         # 任务安装脚本
├── run_now.bat           # 立即执行脚本
├── app.py                # Web应用
├── import_data.py        # 手动导入工具
├── data/                 # 爬取的数据存放目录
│   └── archive/          # 已处理的数据归档
├── logs/                 # 日志文件目录
│   ├── crawler_YYYYMMDD.log
│   ├── import_YYYYMMDD.log
│   ├── scheduler_YYYYMMDD.log
│   └── report_*.txt
└── faam_products.db      # SQLite数据库
```

## 安装步骤

### 方法1: 使用安装脚本(推荐)

1. **右键点击** `setup_task.bat`
2. 选择 **"以管理员身份运行"**
3. 按照提示完成安装

### 方法2: 手动配置任务计划

如果安装脚本无法运行,可以手动配置:

1. 打开 **任务计划程序** (taskschd.msc)
2. 点击右侧 **"创建基本任务"**
3. 填写任务信息:
   - 名称: `FAAM_AutoCrawler`
   - 描述: `FAAM自动化爬虫系统`
4. 触发器设置:
   - 选择 **"每天"**
   - 开始时间: `18:30:00`
5. 操作设置:
   - 操作: **"启动程序"**
   - 程序: `python`
   - 参数: `"D:\Lingma\mission0415-\FAAM\auto_scheduler.py"`
   - 起始于: `D:\Lingma\mission0415-\FAAM`
6. 完成前勾选 **"当单击'完成'时,打开此任务属性的对话框"**
7. 在属性窗口中:
   - 勾选 **"使用最高权限运行"**
   - 选择 **"不管用户是否登录都要运行"**

## 使用方法

### 查看任务状态
```bash
schtasks /query /tn "FAAM_AutoCrawler"
```

### 立即执行一次(测试用)
双击运行 `run_now.bat` 或在命令行执行:
```bash
python auto_scheduler.py
```

### 禁用任务
```bash
schtasks /change /tn "FAAM_AutoCrawler" /disable
```

### 启用任务
```bash
schtasks /change /tn "FAAM_AutoCrawler" /enable
```

### 删除任务
```bash
schtasks /delete /tn "FAAM_AutoCrawler" /f
```

### 修改执行时间
1. 打开任务计划程序
2. 找到 `FAAM_AutoCrawler` 任务
3. 右键 -> 属性 -> 触发器 -> 编辑
4. 修改时间为所需时间

## 日志查看

所有日志文件保存在 `logs/` 目录下:

- `crawler_YYYYMMDD.log` - 爬虫日志
- `import_YYYYMMDD.log` - 导入日志
- `scheduler_YYYYMMDD.log` - 调度器日志
- `report_YYYYMMDD_HHMMSS.txt` - 统计报告

### 查看最新日志
```bash
# Windows PowerShell
Get-Content logs\scheduler_*.log -Tail 50

# 或者直接用记事本打开
notepad logs\scheduler_20260415.log
```

## 常见问题

### Q1: 任务没有按时执行?
**检查:**
1. 任务是否启用: `schtasks /query /tn "FAAM_AutoCrawler"`
2. 查看日志文件是否有错误
3. 确认电脑在18:30时处于开机状态
4. 检查Python是否在PATH中

### Q2: 如何更改执行时间?
**方法1:** 使用命令
```bash
schtasks /change /tn "FAAM_AutoCrawler" /st 19:00
```

**方法2:** 在任务计划程序中修改触发器时间

### Q3: 爬虫失败怎么办?
1. 查看 `logs/crawler_*.log` 了解错误原因
2. 检查网络连接
3. 手动运行测试: `python auto_crawler.py`
4. 确认Target网站可访问

### Q4: 数据库没有更新?
1. 查看 `logs/import_*.log` 了解导入情况
2. 检查data目录下是否有新的Excel文件
3. 确认Excel文件格式正确
4. 手动运行: `python auto_import.py`

### Q5: 如何备份数据?
定期复制以下文件:
- `faam_products.db` - 数据库文件
- `data/archive/` - 历史数据文件
- `logs/` - 日志文件

### Q6: 电脑关机了怎么办?
任务计划程序只会在电脑开机且到达设定时间时执行。如果错过执行时间:
- 下次到达设定时间会自动执行
- 或者手动运行 `run_now.bat` 立即执行

## 性能优化建议

1. **网络**: 确保稳定的网络连接
2. **存储**: 定期清理旧的日志文件(保留最近30天)
3. **数据库**: 每月优化一次数据库
   ```sql
   VACUUM;
   ANALYZE;
   ```
4. **归档**: data/archive目录的文件可以定期压缩备份

2. **每周检查**: 查看Web平台数据是否正常更新
3. **每月检查**: 清理旧日志,备份数据库
