# FAAM 平台 - 快速开始指南

## 第一步: 安装依赖

打开命令行,进入FAAM目录:

```bash
cd D:\Lingma\mission0415-\FAAM
```

安装Python依赖包:

```bash
pip install Flask pandas openpyxl requests
```

## 第二步: 准备数据

确保你有从爬虫导出的Excel文件,文件应该包含以下列:
- TCIN
- 名称
- 品牌 (A New Day 或 Wild Fable)
- 价格
- 原价
- 图片链接
- 购买链接
- 等其他商品信息

## 第三步: 导入数据

运行数据导入脚本:

```bash
python import_data.py
```

按照提示输入Excel文件所在的目录路径。

## 第四步: 启动应用

### 方法1: 使用启动脚本(推荐)

双击运行 `start.bat` 文件

### 方法2: 手动启动

```bash
python app.py
```

## 第五步: 访问平台

打开浏览器,访问: **http://localhost:5000**

## 主要功能

## 常见问题

### Q: 提示找不到Flask模块?
A: 运行 `pip install Flask` 安装Flask

### Q: 数据库为空?
A: 需要先运行 `import_data.py` 导入Excel数据

### Q: 图片无法显示?
A: 确保商品数据中包含有效的图片URL,或将图片下载到 `images/` 目录

### Q: 如何更新数据?
A: 重新运行爬虫获取最新数据,然后再次运行 `import_data.py` 导入

## 技术支持

如有问题,请检查:
1. Python版本 >= 3.8
2. 所有依赖已正确安装
3. Excel文件格式正确
4. 数据库文件有写入权限
