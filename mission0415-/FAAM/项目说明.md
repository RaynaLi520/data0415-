# FAAM 女装信息平台 - 项目总结

## 项目概述

FAAM (Fashion Analytics & Asset Management) 是一个专为追踪和分析 Target 旗下 A New Day 和 Wild Fable 品牌女装商品而设计的Web平台。

## 已完成功能

### 1. Web应用架构
- Flask后端框架
- SQLite数据库存储
- 响应式前端设计
- RESTful API接口

### 2. 核心页面

#### 首页 (/)
- 商品统计概览(总数、新品数、各品牌数量)
- 今日新品展示
- 快速搜索入口

#### 商品查询 (/products)
- 多条件筛选(品牌、新品状态)
- 关键词搜索
- 分页浏览
- 商品卡片展示(图片、价格、评分、颜色、尺码)

#### 每日新品 (/new-arrivals)
- 按日期查看历史新品
- 新品详细信息
- 跳转到Target官网

### 3. 数据管理

#### 数据库设计
- products表: 存储所有商品信息
- daily_new_arrivals表: 记录每日新品
- 索引优化查询性能

#### 数据导入
- import_data.py脚本
- 支持Excel文件批量导入
- 自动处理数据格式

### 4. 用户界面
- 现代化渐变设计
- 响应式布局(支持移动端)
- 商品卡片悬停效果
- 徽章标识(NEW、清仓)
- 友好的空状态提示

## 技术栈

- **后端**: Flask 3.x
- **数据库**: SQLite
- **数据处理**: Pandas
- **前端**: HTML5 + CSS3 (原生)
- **Excel处理**: OpenPyXL

## 数据来源

本平台整合了两个爬虫脚本的功能:

1. **0414代码.txt**: 主要爬虫
   - 爬取Target女装商品
   - 提取详细商品信息
   - 识别新品和清仓状态
   - 下载商品图片

2. **新品代码.txt**: 新品检测
   - 对比历史数据
   - 验证新品资格
   - QA和PDP检查
   - 图片下载

## 使用方法

### 首次使用

1. 安装依赖: `pip install Flask pandas openpyxl`
2. 导入数据: `python import_data.py`
3. 启动应用: `python app.py` 或双击 `start.bat`
4. 访问: http://localhost:5000

### 日常使用

1. 运行爬虫获取最新数据
2. 使用import_data.py导入新数据
3. 在Web平台查看和分析
