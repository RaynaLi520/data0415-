# FAAM 自动化系统 - 快速参考卡片

## 🚀 快速开始

### 首次安装(仅需一次)
```bash
# 右键以管理员身份运行
setup_task.bat
```

### 立即测试
```bash
# 双击运行
run_now.bat
```

## ⏰ 自动执行时间
**每天北京时间 18:30**

## 📊 数据流程
```
爬取Target → 生成Excel → 导入数据库 → 标记新品 → 更新网站
```

## 🔍 常用命令

### 查看任务状态
```bash
schtasks /query /tn "FAAM_AutoCrawler"
```

### 手动执行
```bash
python auto_scheduler.py
```

### 禁用/启用任务
```bash
schtasks /change /tn "FAAM_AutoCrawler" /disable
schtasks /change /tn "FAAM_AutoCrawler" /enable
```

### 修改执行时间(例如改为19:00)
```bash
schtasks /change /tn "FAAM_AutoCrawler" /st 19:00
```

### 删除任务
```bash
schtasks /delete /tn "FAAM_AutoCrawler" /f
```

## 📁 重要目录

| 目录 | 说明 |
|------|------|

| 文件名 | 说明 |
|--------|------|

## 🐛 故障排查

**任务未执行?**
1. 检查任务状态: `schtasks /query /tn "FAAM_AutoCrawler"`
2. 查看日志文件
3. 确认电脑在18:30时开机

**爬虫失败?**
1. 检查网络连接
2. 查看 `logs/crawler_*.log`
3. 手动测试: `python auto_crawler.py`

**数据未更新?**
1. 查看 `logs/import_*.log`
2. 检查 `data/` 目录是否有新文件
3. 手动测试: `python auto_import.py`

## 💡 提示

- 所有操作都会记录日志
- 数据文件会自动归档
- Web应用会自动显示最新数据
- 建议每周检查一次日志
