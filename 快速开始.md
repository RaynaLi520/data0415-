## 💡 常用 SQL 查询

在 DBeaver 的 SQL 编辑器中可以执行以下查询：

```sql
-- 查看所有商品
SELECT * FROM products;

-- 查看新品
SELECT tcin, title, brand, price 
FROM products 
WHERE is_new_arrival = 1;

-- 按品牌统计
SELECT brand, COUNT(*) as count, AVG(price) as avg_price
FROM products 
GROUP BY brand 
ORDER BY count DESC;

-- 查找清仓商品
SELECT * FROM products WHERE is_clearance = 'Yes';

-- 高评分商品
SELECT * FROM products 
WHERE rating >= 4.5 AND rating_count > 100
ORDER BY rating DESC;
```

## 📈 数据库表结构
